2. 布局结构

桌面端 >1024px

  • 左侧导航栏（240px 固定宽度）
  • 顶部面包屑 + GitHub 链接
  • 中央内容区（flex:1）
  • 底部版权信息

平板 768-1024px

  • 左侧导航收缩为图标栏（60px）
  • 或汉堡菜单展开

移动端 <768px

  • 顶部汉堡菜单
  • 全屏覆盖式侧栏
  • 底部 Tab 栏（可选）

3. 工具加载机制

Hash 路由 → fetch 工具 HTML → 注入内容区 → 执行 init() → 切换时调用 destroy()

每个工具独立 CSS 作用域（.tool-{name} 前缀），切换时自动清理
